# Welcome

### Welcome to **UCL Institute of Ophthalmology Bioinformatics Hub**!&#x20;

Funded by Moorfields Biomedical Research Centre, **IoO Bioinformatics Hub** aims to encourage researchers with few bioinformatics experience to utilise computational approach in their research endeavours.

### Are you new to bioinformatics?

Command-line interface, complex file formats, and various software might scare you at first, but with detailed guides to use the simple tools, you'll be able to run your analysis yourself.

If you’re still having trouble following the instructions, please feel free to contact us at **73371fa4.live.ucl.ac.uk@emea.teams.ms** or **jiwon.kim@ucl.ac.uk** with a brief description of the issue. We’ll get back to you as soon as possible to assist you.
