# Nightingale nuclear magnetics resonance (NMR) #0: Introduction

[Nightingale Health](https://research.nightingalehealth.com/coremetabolomics/blood-analysis-service?gad_source=1\&gad_campaignid=21399497693\&gbraid=0AAAAACoUEGo2CiH3X9LPqLkODvImTfUQ6\&gclid=CjwKCAjwv5zEBhBwEiwAOg2YKF2mkHN92qUwQWO9hcHucpxwIwMkh_fJzFX3y7ACSMhB8DzzCiEptxoCbiwQAvD_BwE) offers a **high‑throughput nuclear magnetic resonance (NMR)**‑based platform that quantitatively profiles up to **\~250 metabolic biomarkers.**&#x20;

These biomarkers span lipid metabolism (including 14 lipoprotein subclasses with their detailed composition), fatty acids, amino acids, ketone bodies, glycolysis-related metabolites, inflammation markers, kidney function (e.g., creatinine), fluid balance, and insulin sensitivity indicators.

You can use R packages `tidyverse` and `ggforestplot` to import and visualize your data.
